# Stu_labnet
用于汕头大学实验室计算机能够自动每天连接服务器

detect_link用于检测校园网是否连接 如果没有连接校园网那么会自动登录校园网账号
同时为每天能够有效自动定时检测校园网是否有效开启 将该脚本添加到任务计划程序中
